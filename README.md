# Movie Tracker

Приложение для отслеживания фильмов которые хочешь посмотреть или уже посмотрел.

## Как запустить

```
npm install
npm run dev
```

После этого открыть http://localhost:5173

## Что умеет

- добавлять фильмы с названием, жанром и оценкой
- отмечать просмотренные
- редактировать и удалять
- фильтровать по жанру
- поиск по названию
